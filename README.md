# Prompt2

Prompt2 is a functional GitHub Pages builder for reusable AI creation workflows.

Current version: v3.0.0

- DONE: Standardized page structure
- DONE: Interactive feature cards
- DONE: RBNO builder

- index.html

- data/templates.js

## Usage

1. Open the GitHub Pages site.
2. Pick a type.
3. Paste a rough idea.
4. Click Convert.
5. Copy, save or export the result.

## Version Rule

- Major functional rebuild: +1.0
- Minor feature update: +0.1
- Small copy or style fix: +0.0.1
