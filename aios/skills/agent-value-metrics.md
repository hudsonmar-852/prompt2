# SK-010 — Agent Value and Production Metrics

## Purpose
Measure approved business and production outcomes instead of content volume.

## Core Metrics

- first-pass acceptance rate
- approved artifact count
- human rework rate
- rollback rate
- cost per approved artifact
- time to approval
- reusable asset rate
- provider success rate
- image regeneration rate and average attempts
- cost per merged change for development workflows

## Rules

Metrics must link to runtime, artifact, provider/model, approval and provenance records. Prompt count, token count and generated-file count may be diagnostic signals but are not primary success measures.
